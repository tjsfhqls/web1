인류가 사용하는 모든 데이터가 담기는 그릇인 Web을 제작하는 HTML
PUBLIC DOMAIN: 저작권이 없는 도메인 -> 우리가 만든 웹 페이지를 여러 브라우저에서 똑같이 볼 수 있게 된 것은 바로 이러한 이유 (거대한 바다, 저작권 and 특허는 섬)

태그(Tag) <>

strong ~ /strong = 글씨 두껍게
u ~ /u = 밑줄
h1 ~ /h1 = 제목, 줄바꿈 됨 (6까지 있으며, 커질수록 글씨 크기가 작아짐)
br = 줄 바꿈
p ~ /p = 단락 나누기 (paragraph) * 웹 페이지를 정보로써 좀 더 가치있게 만듬

img = 이미지 source = src list(li) = 제목 > ul(unordered list) = 부모태그 ol(ordered list) = 자동으로 맨 윗줄이 numbering (l = list) a(anchor) = 정보의 바다에 정박한다는 시적의미인 닻 > link 링크 만드는 방법 = href, 새 탭으로 열기 = _blank
